### description

A simple Portfolio template for developer/designers built with React. 

### [live preview](https://coder-nisha.herokuapp.com/)

![react portfoiio](src/assets/images/my-portfolio.gif)


### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place



### Setup

Get the code.

 <pre>git clone https://github.com/codernisha/my-portfolio.git</pre>
 
Install required dependencies

<pre>yarn install</pre>

modify pages content in  `src/content_option.js`.

statrt the server

<pre>yarn start</pre>
